6.1.4 ODATA-880 - 14.5.10: edm:Null can have edm:Annotation child elements

Ralf: ODATA-880 is OPEN

Ralf:14.5.10 _expression_ edm:Null
The edm:Null _expression_ returns an untyped null value. The null _expression_ MUST NOT contain any other elements or expressions. The null _expression_ MUST be written with element notation.
Example 65:

 
<Annotation Term="org.example.display.DisplayName">
  <Null/>
</Annotation>

Mike: alternate wording:
"The only allowed child element of the null _expression_ is an edm:AnnotationType element.".

Mike: I move we resolve ODATA-880 by replacing the current sentence prohibiting child elements of the null _expression_ with the following text: "The only allowed child element of the null _expression_ is an edm:Annotation element.". Mark seconds

ODATA-880 is resolved as proposed
