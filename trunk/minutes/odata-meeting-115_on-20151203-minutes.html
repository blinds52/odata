5. JSON Format for CSDL

Approve OData JSON Format for CSDL as CSD01 and initiate public review

Susan: I move that the TC approve OData JSON Format for Common Schema Definition Language (CSDL) Version 4.0 Working Draft 01 and all associated artifacts packaged together in https://www.oasis-open.org/apps/org/workgroup/odata/download.php/57012/odata-json-csdl-v4.0-wd01-2015-11-20-final.zip as a Committee Specification Draft 01 and designate the Microsoft Word version of the specification as authoritative.

Susan: I further move that the TC approve submitting the aforementioned Committee Specification Draft 01 for 30 days of public review and notifying all external stakeholders known to the members of the TC. Ramesh seconds.

The motion is approved unanimously.

7. OData international standardization status

ISO/IEC DIS 20802-1: 3720. Information technology -- Open data (OData) protocol -- Part 1: Protocol plus errata 02
ISO/IEC DIS 20802-2: 3721. Information technology -- Open data (OData) protocol -- Part 2: JSON format version 4.0 plus errata 02
The Draft International Standard (DIS) ballot to approve the OData v4 as an international standard is on (Nov 2015 through Jan 2016).
The eventual approved international standard would have two parts (OData v4 Core and OData JSON Format v4).
